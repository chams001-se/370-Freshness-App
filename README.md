# 370-Freshness-App
Made by Jeremy Molina, Nicholas Stern, Sam Chamsayidi, and Kyle Billan
ShelfLife is a Java Swing application made over the span of August - December 2025 for a project in SE-370. 
Run *Main.java* to begin.
To use it you can select a date on the calendar then press the + button located in the bottom right of the calendar. This will allow you to enter the food product name and quantity.
You can edit and remove this entry if desired.

To learn more about the design process, see our SRS document: https://docs.google.com/document/d/1ME8BEjUlpk47Lf4n3WlSn-6AxDqEzTaLoOo-WxSSZwo/edit?usp=sharing
